# AI Agent Instructions

This repository contains a frontend website project with a CMS-driven blog and event calendar.

AI agents modifying this project should follow the rules below.

---

## Tech Stack

- HTML
- SCSS
- Vanilla JavaScript
- CMS used for blog posts and event calendar content

Avoid introducing frameworks unless explicitly requested.

---

## Code Style

HTML

- Use semantic HTML elements
- Ensure accessibility
- Use descriptive alt attributes for images

CSS / SCSS

- Follow BEM naming conventions
- Use modular SCSS structure
- Avoid inline styles
- Prefer reusable classes

JavaScript

- Use modern ES6+ syntax
- Keep scripts modular
- Avoid global variables

---

## Accessibility

- Ensure keyboard navigation
- Use ARIA attributes where necessary
- Maintain proper heading hierarchy
- Ensure sufficient color contrast

---

## Performance

- Optimize images
- Prefer SVG icons
- Avoid large JavaScript bundles
- Keep dependencies minimal

---

## Project Structure

src/
assets/ images, icons, fonts
styles/ SCSS files
js/ JavaScript modules
components/ reusable components

public/
static HTML entry points

design/
mockups and design materials

tasks/
development tasks used by AI and developers

---

## Development Guidelines

- Prefer reusable components
- Avoid duplication
- Keep code simple and readable
- When generating code, modify existing files instead of creating unnecessary new ones

---

## CMS Integration

- Blog content is managed through the CMS
- Event calendar entries are managed through the CMS
- Generated code should allow dynamic content integration
